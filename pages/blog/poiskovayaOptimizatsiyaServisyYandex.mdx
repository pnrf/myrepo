# Поисковая оптимизация. Сервисы Яндекса

*Это дорожная карта по оптимизации сайта для поисковой выдачи Яндекса.*  
*Если вы не знакомы с данной темой, советую предварительно посмотреть [обучающие материалы ↗](https://yandex.ru/adv/edu/materials/tag-webmaster?from=webmaster-help).*

---

## Шаг 1. Яндекс ID 

Зарегитсрируйтесь в [Яндекс ID ↗](https://yandex.ru/id). Этим вы настроите один аккаунт для всех его сервисов.

## Шаг 2. Вебмастер

2.1. Перейдите в [Яндекс Вебмастер ↗](https://webmaster.yandex.ru/welcome/) и добавьте свой сайт. 

При добавлении сайта потребуется подтвердить ваши права на него. Сделайте это. Руководствуйтесь подсказками и [справкой ↗](https://yandex.ru/support/webmaster/index.html).

Первоначальная индексация сайта занимает время (до 2-ух недель). Учитывайте это заранее.
